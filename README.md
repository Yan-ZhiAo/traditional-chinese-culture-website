# 宣传弘扬中华优秀传统文化

宣传弘扬中华优秀传统文化是一个面向传统文化主题展示的静态网页课程设计项目。项目围绕中华优秀传统文化的精神内涵、文化鉴赏、知识简介和当代表达展开，使用 HTML、CSS、JavaScript 与 jQuery 构建多页面浏览体验，并结合图片瀑布流、首屏视频、滚动交互和图文排版展示传统文化的历史底蕴与传承价值。

## 项目功能

- 理论论述：通过首屏视频、轮播内容和主题图文，展示中华优秀传统文化的精神命脉、创造性转化与民族共同体意识。
- 文化鉴赏：使用图片瀑布流展示传统文化素材，支持点击图片放大预览。
- 文化简介：从综合性、和谐性、包容性、儒家思想、传统节日和文化分类等角度介绍中国传统文化。
- 文化传承：结合传统文化出圈案例、传统节日自述和滚动视觉效果，呈现传统文化在当代传播中的生命力。
- 登录页面：提供与整站视觉风格一致的登录界面页面，用于完善课程设计页面结构。
- 静态入口：`index.html` 会自动跳转到主题首页 `Discussion.html`，便于本地打开和后续部署。

## 快速开始

本项目是纯静态网页，不需要安装依赖。

### 1. 直接打开

双击 `index.html`，或使用浏览器打开 `Discussion.html` 即可进入网站。

### 2. 使用本地静态服务

如果希望通过本地服务访问，可以在项目目录运行：

```powershell
python -m http.server 5500
```

然后访问：

```text
http://localhost:5500
```

## 页面说明

| 页面 | 说明 |
| --- | --- |
| `Discussion.html` | 网站首页，包含首屏视频、主题轮播、理论论述和民族共同体意识相关内容 |
| `Appreciation.html` | 文化鉴赏页面，通过 JavaScript 动态生成 47 张图片的瀑布流展示 |
| `Introduction.html` | 文化简介页面，系统介绍中国传统文化的特点、内容分类和价值判断 |
| `Smriti.html` | 文化传承页面，展示传统文化当代表达、传播案例和传统节日主题文字 |
| `Login.html` | 登录视觉页面，用于补充完整网站导航与交互入口 |

## 素材说明

项目使用本地图片、字体、GIF 和视频素材构成完整视觉体验，其中 `image/synth city by VISUALDON 4K 60 FPS.mp4` 文件体积较大，已通过 Git LFS 跟踪。克隆仓库后如需完整视频素材，请确保本机安装 Git LFS 并执行：

```powershell
git lfs pull
```
